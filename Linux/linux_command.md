# Linux 命令
[Linux 命令大全](http://man.linuxde.net/)

## 常用

| 命令 | 说明 |
| :------ | :------ |
| mv | 移动文件/目录，重命名 |
| cp | 复制文件/目录 |
| find | 查找文件 |
| locate | 查找文件 |
| head | 显示文件前10行(-n 可指定显示多少行）|
| tail | 显示文件后10行文件(-n 可指定显示多少行) |
| mkdir | 创建目录 |
| touch | 创建文件 |
| chomd | 更改文件或者目录的全选 |
| gzip | 压缩文件 |
| gunzip | 解压缩文件 |
| tar | 打包/解包 |
| which/whereis | 查找命令所在路径 |
