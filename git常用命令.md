# Git常用命令

## 合并多个commit

### git rebase

#### 从 HEAD 版本开始往过去数3个版本

	git rebase -i HEAD~3
	
#### 指名要合并的版本之前的版本号


	git rebase -i 3a4226b 		#请注意3a4226b这个版本是不参与合并的，可以把它当做一个坐标
	
执行了rebase命令之后，会弹出一个窗口

	pick 3ca6ec3   '注释**********'
	
	pick 1b40566   '注释*********'
	
	pick 53f244a   '注释**********'

将pick改为squash或者s,之后保存并关闭文本编辑窗口即可

	pick 3ca6ec3   '注释**********'
	
	s 1b40566   '注释*********'
	
	s 53f244a   '注释**********'
